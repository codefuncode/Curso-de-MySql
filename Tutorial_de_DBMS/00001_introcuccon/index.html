Introducción a DBMS Primero vemos que es una base de datos. En informática, una base de datos es una colección organizada de datos almacenados y a los que se accede electrónicamente desde un sistema informático. Cuando las bases de datos son más complejas, a menudo se desarrollan utilizando técnicas formales de diseño y modelado. Ahora veamos que es un DBMS. Los DBMS por sus siglas en inglés significa Database Management System. Podemos romperlo así DBMS = Base de datos + Sistema de gestión. La base de datos es una colección de datos y el sistema de gestión es un conjunto de programas para almacenar y recuperar esos datos. Basándonos en esto, podemos definir DBMS así: DBMS es una colección de datos interrelacionados y un conjunto de programas para almacenar * acceder a esos datos de una manera fácil y efectiva.
